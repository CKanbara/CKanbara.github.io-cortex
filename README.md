# CORTEX – 文章お手軽修正マシン

CORTEX は、日本語文章の表記揺れや形式的な修正を、  
ルールベースで手軽に行うことを目的として開発された Web ツールです。

- 形式段落の字下げ
- 三点リーダの個数統一
- 全角／半角の整理
- 句読点・記号まわりの調整  
など、一般的なテキストエディタでは少々面倒な修正を、  
チェックボックス操作だけで行えることを目指していました。

---

## このリポジトリについて

本リポジトリは、**2019年から約7年間運用していた Web サービス「CORTEX」 の静的アーカイブ**です。

2026年3月31日をもって Web サービスとしての提供は終了し、  
以降は **GitHub Pages 上で動作する静的版**として公開しています。

- サーバーサイド処理（PHP・ログイン・保存機能）は廃止
- すべてクライアントサイド JavaScript で完結
- 個人情報の送信や保存は一切行いません

「かつて動いていたサービスの記録」として、  
また、ルールベース文章整形ツールの一例として残すことを目的としています。

---

## 公開版の仕様（GitHub Pages 版）

- ログイン機能：❌ なし
- 会員区分（無料／プレミアム）：❌ 廃止
- 文章・設定の保存：ブラウザ内（localStorage のみ）
- 利用制限：なし
- 対応環境：モダンブラウザ（Chrome / Edge / Firefox / Safari）

※ 旧サービスで提供していた一部機能は削除・簡略化されています。

---

## なぜサービスを終了したのか

近年、文章生成 AI や文章校正支援ツールが広く普及し、  
本サービスが提供していた機能の多くは、  
より高度かつ汎用的な手段で代替可能になりました。

その状況を踏まえ、  
**CORTEX は一定の役割を終えた**と判断し、  
Web サービスとしての運営を終了しました。

---

## 技術的メモ

- 元実装：PHP + JavaScript
- 現在：HTML + JavaScript（完全静的）
- 文章整形ロジックは初期実装当時のまま保持しています
- 不要になったスクリプト・広告コードは整理・削除しています
